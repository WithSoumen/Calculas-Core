# 📐 Calculas Core

A high-performance, web-native graphing engine engineered for advanced mathematical functions and real-time numerical calculus visualization.

## 🚀 Features

* **Advanced Graphing:** Plots trig, inverse functions, e^x, logs, and custom user expressions.
* **Instant Derivatives:** Computes and overlays the first derivative $f'(x)$ on the fly.
* **Interactive Integrals:** Evaluates definite integrals ($\int_{a}^{b}$) with dynamic Riemann sum shading.
* **Tangent Tracker:** Projects instantaneous slopes and tangent lines that track your mouse cursor.
* **Pro UI:** Responsive, high-contrast dark theme with fluid panning and scroll-wheel zooming.

## 🛠️ Tech Stack

* **HTML5** (Canvas API for high-performance raster rendering)
* **CSS3** (Modern dark-mode UI layout)
* **Vanilla JavaScript** (Numerical calculus parsing and matrix math engine)
